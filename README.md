# Homebrew Tap for kraze

Homebrew tap for [kraze](https://github.com/hjames9/kraze) - a Kubernetes development environment manager with docker-compose-like experience.

## Installation

```bash
brew install hjames9/kraze/kraze
```

## Requirements

kraze requires a Docker-compatible runtime (Docker Desktop, Colima, Podman, or Rancher Desktop).

## Documentation

See the [main repository](https://github.com/hjames9/kraze) for complete documentation and usage examples.
